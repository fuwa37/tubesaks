# Aplikasi Kaca & Kusen
Prototype aplikasi yang menghitung kebutuhan kaca dan/atau kusen untuk dipotong menjadi bagian-bagian kecil dan menghitung kebutuhan pekerja untuk melaksanakan pemotongan. Histori pemotongan akan disimpan dalam suatu basis data yang dapat dilihat kembali. Histori dapat disimpan ke dalam bentuk .csv

Aplikasi dibangun menggunakan Java
